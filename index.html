FeaturePopups

Run the proposal

This is a sandbox to discuss the implementation of FeaturePopups control in OpenLayers

You can run this proposal in:

  • feature-popups.html

Features

This control wraps the management of popups and SelectFeature control.

  • Prepare the contents of the popup using templates
    • Templates as strings or functions
    • Allows external templates renderers
  • Show popups by selection or hover from multiple vector layers.
  • Multiple selection using box and show list of features selecteds into popup.
  • Allows to register listeners on the events of selection and hover

The beginnings

This development began with the request olsocial of Javier Carrasco, but has gone far beyond its initial request.
