OpenLayers Example

popups, vector layers, templates

Demonstrates how to use the "FeaturePopups" control to display popups on multiple vector layers using templates.

This control wraps the management of popups and SelectFeature control.

Features:

  • Prepare the contents of the popup using templates (as strings or functions)
  • Simultaneously manage popups by selection or hover in multiple vector layers.
  • Allows to register listeners on the events of selection and hover (Advanced)

View the feature-popups.js source to see how this is done.
